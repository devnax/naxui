
### How to use

```tsx
import Container, {ContainerProps} from 'naxui/Container'

const App = () => {
    return (
        <Container {...ContainerProps}>
            container area
        </Container>
    )
}
```

### Props
You can use the html `div` props and and theme `css, aliases props` and also below props.

| Name     | desciption                   |
| -------- | ---------------------------- |
| maxWidth | `number, xs, sm, md, lg, xl` |

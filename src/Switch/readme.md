
### How to use

```tsx
import Switch, {SwitchProps} from 'naxui/Switch'

const App = () => {
    return (
        <Switch {...SwitchProps} /> 
    )
}
```

### Props
You can use the html `input checkbox` props and and theme `css, aliases props` and also below props.

| Name  |  desciption |
|---|---|
|  checked |  `boolean` check and uncheck the Switch|
|  color |  `"primary" | "secondary" | "success" | "error" | "warning"` |
|  size |  `number` you can set Switch size |
